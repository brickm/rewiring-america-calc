Critical Context: Health Benefits

‚ö†Ô∏è Important: Health benefits are REAL but MODEST compared to other interventions

Cost-Effectiveness Comparison

  • Substance use treatment: $10K-25K per life saved
  • Violence intervention: $20K-50K per life saved
  • Food assistance: $30K-100K per life saved
  • Rental assistance: $50K-150K per life saved
  • Home electrification: $200K-600K per life saved

Primary case for electrification: Climate mitigation + energy cost savings. Health benefits are welcome co-benefits, not primary justification.

Equity in Electrification Policy

Who Benefits Most?

Homeowners

More likely white, middle/upper class

Those with upfront capital

Can afford installations even with rebates

Stable housing

Not facing eviction or instability

Equity-Centered Priorities

  • Multi-family buildings (high aggregate impact)
  • Environmental justice areas (worst baseline air quality)
  • Lower-income housing types (mobile homes, small units)
  • Total community impact, not just per-household metrics

Claude Code

Agentic AI coding assistant that works through your terminal

  • Build fully functioning web apps locally or remotely
  • Autonomous coding agent that can plan, write, test, and debug code
  • Works directly in your terminal with your local development environment
  • Understands complex project structures and can make changes across multiple files

Cost: $20/month with Claude Pro subscription

Claude Code democratizes software development, making it accessible to non-programmers while accelerating experienced developers' workflows.

Next.js

React-based web framework for production applications

  • Server-side rendering and static site generation
  • Built-in API routes for backend functionality
  • Optimized performance with automatic code splitting
  • File-based routing system
  • TypeScript support out of the box

Next.js is built and maintained by Vercel, making it the perfect pairing for modern web development and deployment.

GitHub

Version control and collaboration platform

What is .git?

  • A hidden folder that tracks all changes to your code
  • Creates a complete history of your project
  • Allows you to revert to previous versions
  • Enables branching and merging of different code versions

What is GitHub?

  • Cloud-based hosting service for Git repositories
  • Collaboration platform for teams and open source
  • Backup for your code in the cloud
  • Integrates with deployment services like Vercel
  • Issue tracking, pull requests, and code review tools

Git is the tool, GitHub is the platform. Think of Git as Microsoft Word and GitHub as Google Docs.

Vercel

Cloud platform for deploying modern web applications

How Vercel Works

1. Connect: Link your GitHub repository to Vercel

2. Detect: Vercel automatically detects when you push code

3. Build: Runs your build process in the cloud

4. Deploy: Makes your app live at a public URL (~60 seconds)

  • Automatic HTTPS certificates
  • Global CDN for fast loading worldwide
  • Preview deployments for every pull request
  • Easy environment variable management
  • Built by the creators of Next.js

Push to GitHub ‚Üí Vercel builds and deploys automatically. No server configuration needed.

Tailwind CSS

  • Write styles directly in your HTML using utility classes

  • Responsive design with mobile-first approach
